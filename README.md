# Rami Wizard

Projeto gerado com **Vite + React + TypeScript + Tailwind**.

## Rodando localmente

```bash
npm install
npm run dev
# abra http://localhost:5173
```

## Build de produção
```bash
npm run build
npm run preview  # servidor em 4173
```

## Docker (opcional)
```bash
docker build -t rami-wizard .
docker run -p 4173:4173 rami-wizard
# abra http://localhost:4173
```
